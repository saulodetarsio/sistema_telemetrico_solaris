Pasta Dados DSB2020 contém os dados obtidos pela telemetria durante as provas do dsb 2020

Pasta mensagem com timestamp contem modelos de sketches, do sistema de transmissão de dados, que foram sendo desenvolvidos e 
estudados até se chegar no formato final a ser utilizado, que está na pasta mensagem padrão

Pasta mensagem padrão contém o sketche utilizado no arduino para conxão com o mqtt em terra, e transmissão dos dados.
